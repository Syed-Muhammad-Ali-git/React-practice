# React Projects Collection

This repository contains a collection of React projects developed for learning and practice purposes. Each folder represents a separate project with its own setup, features, and documentation.

## Projects

### 1. learning-react-saylani

A practice website built with React using Vite. It includes routing utilities, screen components, and implements various React concepts for hands-on learning.

- **Key Features:**
  - React Router for navigation
  - Utils folder for route management
  - Screen folders acting as pages with JSX components
  - Practice implementation of React concepts

### 2. login_signup_Page_react

A project focused on creating login and register forms using React and MDB React UI Kit. It demonstrates controlled components, validation, and user authentication flows.

- **Key Features:**
  - Controlled inputs for email and password
  - Password show/hide toggle
  - Form validation and localStorage integration
  - Autofocus and keyboard navigation
  - UI Layout with MDB Container, Row, Col
  - Pending: Home page navigation after login, inline error messages

### 3. react-typescript

A minimal setup for React with TypeScript and Vite, including HMR and ESLint rules. Ideal for starting TypeScript-based React projects.

- **Key Features:**
  - TypeScript integration
  - Vite for fast development
  - ESLint configurations for type-aware linting
  - Optional React Compiler setup
  - Support for SWC and Babel plugins

### 4. react-yt-prac

A basic React + Vite template for practicing React concepts, following YouTube tutorials or general learning resources.

- **Key Features:**
  - Minimal Vite setup for React
  - ESLint rules for code quality
  - Suitable for quick prototyping and learning
  - Includes components like navbar, props, and pages for counter, home, inputfield, skills, userdetail

## Getting Started

Each project has its own `README.md` file with detailed setup instructions. Navigate to the respective folder and follow the guidelines to run the project locally.

For example:

- `cd learning-react-saylani && npm install && npm run dev`
